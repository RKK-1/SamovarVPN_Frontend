Samovar VPN — фронтенд

## О продукте

Samovar VPN — это сервис с упором на:
- **Стабильную работу в РФ**: отдельный сервер в белом списке российских провайдеров.
- **Европейские локации** (Швеция, Финляндия, Германия) для низкого пинга и высокой скорости.
- **Простой онбординг** через Telegram‑бота и понятный личный кабинет.

Текущий репозиторий содержит SPA на **React + TypeScript + Vite**, в котором реализованы:
- **Лендинг** с тарифами и блоком локаций.
- **Страница авторизации** (email / Telegram) с валидацией и редиректами.
- **Личный кабинет** с отображением статуса подписки, ключей и устройств.

## Требования

- Node.js 18+ (рекомендуется LTS).
- npm (идёт вместе с Node.js).

## Установка и запуск

```bash
cd samovar-frontend

# Установить зависимости
npm install

# Запустить dev-сервер
npm run dev
```

По умолчанию Vite поднимет фронтенд по адресу:

- `http://localhost:5173/`

## Сборка на прод

```bash
cd samovar-frontend
npm run build
```

Собранный статику Vite положит в папку `dist/`, её можно отдавать любым HTTP‑сервером (Nginx, Caddy и т.д.).

## Настройка API

Базовый URL бэкенда задаётся через переменную окружения:

- в файле `.env`:

```bash
VITE_API_BASE_URL=http://127.0.0.1:8080/api
```

Фронтенд обращается к бэкенду через этот URL для:
- авторизации (`/register`, `/login`, `/login-by-telegram*`);
- платежей (`/create-payment`);
- кабинета (`/check-*`, `/get-license-*`, `/devices`, `/renew-link`).
